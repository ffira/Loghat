## 🚀 One-Click Web Deployment

Click the **Deploy to Render** button above to automatically:
1. Provision a free PostgreSQL database.
2. Link the database to the web server container.
3. Generate secure tokens.

You will only need to input your Stripe and Gemini API keys in the provided fields on Render.

For a full publishing guide (including setting up your native iOS Mobile App), see our **[Publishing Playbook](.gemini/antigravity/brain/dcff6325-5520-4df6-9931-d2e521d26432/publishing_playbook.md)**.

## 💻 Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
